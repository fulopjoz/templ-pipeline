Function Reference Table

Comprehensive reference of all functions used in the MCS workflow, organized by functional category for easy navigation and understanding.

Core MCS Functions

  • find_mcs(): Primary MCS finding with progressive threshold reduction
  • get_central_atom(): Find central atom by shortest path sum
  • find_best_ca_rmsd_template(): Select template with best CA RMSD
  • rdRascalMCES.FindMCES(): RascalMCES algorithm for MCS detection

Conformer Generation

  • constrained_embed(): Generate conformers with MCS constraints
  • central_atom_embed(): Fallback embedding using central atom
  • embed_with_uff_fallback(): UFF-compatible embedding for organometallics
  • rdDistGeom.EmbedMultipleConfs(): RDKit conformer generation

Molecular Validation

  • validate_molecular_connectivity(): Check for molecular fragmentation
  • validate_molecular_geometry(): Validate bond lengths and geometry
  • log_coordinate_map(): Debug coordinate constraint details
  • relax_close_constraints(): Remove constraints too close together

Force Field Optimization

  • simple_minimize_molecule(): Unconstrained MMFF/UFF minimization
  • validate_mmff_parameters(): Check MMFF parameter availability
  • needs_uff_fallback(): Detect organometallic atoms
  • AllChem.MMFFOptimizeMolecule(): MMFF force field optimization

Molecular Alignment

  • rdMolAlign.AlignMol(): Post-embedding molecular alignment
  • Chem.GetSubstructMatch(): SMARTS pattern matching
  • Chem.MolFromSmarts(): Convert SMARTS to molecular pattern
  • Chem.AddHs(): Add hydrogens with coordinates

Error Handling

  • UnifiedErrorTracker: Centralized error tracking system
  • ErrorCategory: Categorized error types
  • ErrorSeverity: Error severity levels
  • safe_name(): Filesystem-safe name conversion

Constants Configuration

  • N_CONFS = 50: Default number of conformers
  • MIN_PROTEIN_LENGTH = 20: Minimum protein length
  • DEFAULT_MMFF_ITERATIONS = 500: MMFF optimization iterations
  • CA_RMSD_THRESHOLD = 2.0: CA RMSD quality threshold

Advanced Features

  • Progressive threshold reduction: 0.9 to 0.0 in 0.1 steps
  • Organometallic detection: Automatic UFF fallback
  • Memory management: Thread limiting and resource control
  • Quality control: Multi-stage validation checks

Workflow Features Quality Control

  • Progressive Threshold Reduction: RascalMCES with 0.9 to 0.0 similarity threshold
  • Central Atom Fallback: Single atom positioning when MCS fails
  • Constrained Embedding: Lock MCS atoms to template coordinates
  • Geometry Validation: Bond length checks 0.5-3.0Å range
  • Force Field Optimization: MMFF with UFF fallback for organometallics
  • Memory Management: Thread limiting and resource control
  • Error Tracking: Comprehensive error categorization and logging
