Function Reference

Core MCS

  • find_mcs(): RascalMCES-only, thresholds 0.9→0.0; returns best SMARTS or central atom fallback.
  • rdRascalMCES.FindMCES: Finds MCES and SMARTS using Rascal options.
  • get_central_atom(): Chooses atom with minimal sum of shortest-path distances.
  • find_best_ca_rmsd_template(): Picks template with lowest ca_rmsd property.

Embedding

  • constrained_embed(): Builds coordMap from MCS, relaxes constraints, ETKDGv3, progressive reduction, alignment, optional minimization. Falls back for invalid matches or large targets (>150 atoms).
  • central_atom_embed(): Unconstrained embed with UFF fallback, translate to ref central atom; optional minimization.
  • embed_with_uff_fallback(): Tries standard ETKDG, then UFF-friendly settings, then relaxed/minimal settings; caps threads at 3.

Validation

  • validate_molecular_geometry(): Bond-length checks and stats; flags 0.5–3.0 Å outliers.
  • validate_molecular_connectivity(): Fragmentation and coordinate sanity checks.
  • log_coordinate_map() and relax_close_constraints(): Diagnose/mitigate tight constraints.

Optimization

  • simple_minimize_molecule(): MMFF via rdForceFieldHelpers.MMFFGetMoleculeForceField(...).Minimize() with UFF fallback (AllChem.UFFOptimizeMolecule).
  • validate_mmff_parameters() and needs_uff_fallback(): Detect when MMFF is unavailable or organometallics present.

Utilities

  • N_CONFS = 200: Default conformer count.
  • safe_name(): Generates filesystem-safe names.
